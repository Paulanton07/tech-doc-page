Introduction to Electron

Electron is a framework for building cross-platform desktop applications using web technologies.

It combines Chromium and Node.js into a single runtime.

  • Open source
  • Cross-platform
  • Active community
const { app, BrowserWindow } = require('electron');
Setup and Installation

To get started, install Electron via npm:

Make sure you have Node.js installed first.

  • Install Node.js
  • Run npm install electron --save-dev
npm install electron --save-dev
Main vs Renderer Process

Electron apps have two types of processes: main and renderer.

The main process manages the app lifecycle, while renderer processes handle UI.

  • Main process: Node.js environment
  • Renderer process: Browser environment
// main.js // renderer.js

Packaging and Distribution

Package your app using tools like electron-packager or electron-builder.

Distribute your app for Windows, macOS, and Linux.

  • Use electron-packager
  • Use electron-builder
npx electron-packager .
Best Practices and Security

Follow security guidelines to protect your users.

Keep dependencies up to date and avoid using Node.js integration in renderer unless necessary.

  • Disable Node.js integration in renderer
  • Use context isolation
  • Validate user input
webPreferences: { nodeIntegration: false, contextIsolation: true }
